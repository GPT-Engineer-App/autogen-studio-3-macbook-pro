## Setup

```sh
git clone https://github.com/GPT-Engineer-App/autogen-studio-3-macbook-pro.git
cd autogen-studio-3-macbook-pro
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
